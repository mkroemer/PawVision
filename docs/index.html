‚ö° Quick Installation

Get PawVision up and running on your Raspberry Pi with a single command:

curl -sSL https://raw.githubusercontent.com/yourusername/pawvision/main/install.sh | bash

This will automatically:

  • Install all required dependencies
  • Download the latest PawVision
  • Preserve your existing videos and settings
  • Set up the web interface on port 5000

After installation, access the web interface at http://your-pi-ip:5000
